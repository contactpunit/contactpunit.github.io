Functions in Javascript

Posted on Mon 15 June 2020 in Javascript • 1 min read

Functions are first class citizens

Functions are first class citizens in Javascript just like Python. What does this mean ? Well this means functions should obey below rules to be treated as first clas citizens:

  • Functions can be created as literals
function hello() {}
  • Functions can be assigned to variables
var hello = function() {}
  • Functions can be passed to other functions and can be returned as a return value
function test(hello) {
  return hello()
}
function hello() {};
result = test(hello)
  • Function can be assigned properties
var f = function() {};
f.alternateName = 'demo'
